<Section icon={Swords} title="戦闘スタイルと戦闘メカニクスのマスター">

戦闘スタイルは、攻撃能力の核心です。ドロップ率1%のレジェンダリー（Legendary）スタイルも存在しますが、現在はその実用性と使いやすさから、エピック（Epic）ティアの**ボクシング（Boxing）**と**ハケリ（Hakery）**が対戦シーンを支配しています。すべてのスタイルには、無敵フレーム（I-frame）やガード崩しなど、特定の特典があります。

**主要な戦闘コンセプト：**
- **クラッシュ / グラップル（競り合い）**: 2人のプレイヤーが同時に攻撃すると、グラップル状態に入ります。最初に反応したプレイヤーが押し出しに勝ち、相手の防御を無視してダメージを与えます。
- **ブロック削り（チップダメージ）**: 相手がガードしていても、特定のスタイルは「削り」ダメージを与えます。ブロック削りの割合が高いスタイルは、防御的なプレイヤーを素早く削り取ることができます。
- **ガード崩し（ガード貫通）**: 相手のガード防御の一部を無視する攻撃です。

| 戦闘スタイル | レア度 | 主な利点 | 最適な用途 |
| :--- | :--- | :--- | :--- |
| **ボクシング (Boxing)** | **エピック** | 中断されないM2（強攻撃）/ 無敵フレーム | 攻撃的な殴り合い |
| **ハケリ (Hakery)** | **エピック** | 高いガード貫通力 / 素早いM1（弱攻撃） | 亀ガード（防御一辺倒）の打破 |
| **レジェンダリー** | **1%ドロップ** | 絶大なダメージ / 固有スキル | トップ層のリーダーボード対戦 |
| **ベーシック** | **コモン** | バランス型 / 特殊効果なし | クラッシュを学ぶ初心者向け |

<Callout type="info" title="スタイルの相乗効果">
**高身長**のキャラクターモデルと**ボクシング（Boxing）**スタイルの組み合わせは、非常に強力なコンボです。高身長モデルのリーチと、中断されないM2攻撃が合わさることで、効果的なダメージトレード（殴り合い）で敵なしの状態になります。
</Callout>
