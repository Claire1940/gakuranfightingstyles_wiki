<Section icon={Swords} title="格闘スタイルティアリストとパーク">

学ランの核心は、その多様な格闘スタイルにあります。各スタイルには、キャラクターメニューで確認できるユニークなパーク（特典）が備わっています。スタイルはレア度によって分類されており、プレイヤーはより格上のスタイルを狙ってRobuxを消費してリロールすることがよくあります。現在の2026年メタでは、身体の身長と選択したスタイルのシナジーが、アリーナでの強さを決定づけます。

<CardGrid cols={3}>
  <Card title="明/アカリ (Sティア)">
    - **最高射程**: M1攻撃で圧倒的なリーチを誇る
    - **高ダメージ**: 強力なクリティカルフィニッシャー
    - **メタコンボ**: 高身長キャラクターと完璧にマッチする
  </Card>
  <Card title="ムエタイ (Aティア)">
    - **高速攻撃**: 連射のような素早い打撃
    - **脚技重視**: キックを駆使した多様な攻撃判定（ヒットボックス）
    - **スタミナ**: 適度な消費率
  </Card>
  <Card title="ボクシング (Bティア)">
    - **素早いジャブ**: 高速かつ短射程
    - **容易な受け流し（パリィ）**: 防御型プレイヤーに最適
    - **確かな基礎**: タイミングを学ぶのにおすすめ
  </Card>
</CardGrid>

上位ティア以外にも、**空手**、**レスリング**、**ベーシック（基本）**といったスタイルが代替のプレイスタイルを提供します。レスリングは組み技と圧倒的な力で相手のガードを崩すことに特化しており、空手は攻撃と防御のバランスが取れたアプローチを提供します。

| 格闘スタイル | 主なパーク | 推奨ビルド | レア度 |
| :--- | :--- | :--- | :--- |
| **明（アカリ）** | 優れたM1射程 | 最大身長 / 長いリーチ | レジェンダリー |
| **ムエタイ** | 打撃速度 | バランスの取れた身長 / スタミナ | レア |
| **空手** | カウンターバフ | 平均的な身長 | アンコモン |
| **レスリング** | ガードブレイク | 最大身長 | レア |
| **ボクシング** | 素早いジャブ | 低身長 / 高スタミナ | コモン |
| **ベーシック** | 特殊なパークなし | 初心者向けトレーニング | スターター |

<Callout type="tip" title="プロメタのヒント">
もし**明（アカリ）**を引き当てたら、身長のリロールを最優先してください。高身長のアカリ使いは、他のほとんどのスタイルがジャブすら当てられない距離から相手を攻撃できるため、開けた場所での戦闘において圧倒的な戦術的優位性を築くことができます。
</Callout>

<Section icon={Target} title="高度な戦闘メカニクスとコンボマスター">

戦闘における**学ラン完全ガイド**をマスターするには、単にマウスをクリックするだけでは不十分です。このゲームには、ボタン連打（スパム）を咎める洗練されたパリィ（受け流し）とガードシステムが採用されています。単にM1を連打しているだけでは、熟練した相手に簡単にパリィ（Fキー）を合わされ、壊滅的なカウンター攻撃の隙を晒すことになります。

<Steps>
  <Step num="1" title="戦闘モードを有効にする">
    **Tキー**を押して戦闘構えに入ります。キャラクターの姿勢が変わり、アクションの準備が整ったことを示します。
  </Step>
  <Step num="2" title="パリィを極める">
    相手が攻撃モーションを開始するのを待ちます。攻撃が当たる直前に**Fキー**を押してパリィを発動させ、相手を一時的にスタン（気絶）させます。
  </Step>
  <Step num="3" title="基本コンボを実行する">
    最も効果的な基本コンボは **M1 -> M1 -> クリティカル** です。3撃目にクリティカルヒットを繰り出すことで、相手がブロックしようとした場合にそのガードを崩しやすくなります。
  </Step>
  <Step num="4" title="機動力を活用する">
    **Qキー**を押してダッシュします。ダッシュは間合いを詰めたり、スタミナが低下した際に後退したりするのに使用できます。決して一箇所に長く留まらないようにしてください。
  </Step>
</Steps>

間合いの管理（スペーシング）が最後のパズルのピースです。M1攻撃を繰り出す際は、クリックの間にわずかなディレイを挟む「フェザリング（ゆさぶり）」を行う必要があります。これにより、攻撃のリズムが予測不可能になり、相手がパリィを合わせるのが非常に困難になります。常にパリィされてしまう場合は、クリックが早すぎて行動がパターン化している可能性が高いです。

| アクション | キー割り当て | 戦術的な用途 |
| :--- | :--- | :--- |
| **弱攻撃** | M1 (左クリック) | 削りダメージおよびコンボの起点 |
| **クリティカルヒット** | 重攻撃キー | ガード崩しおよびフィニッシャー |
| **パリィ / ブロック** | Fキー | 防御のタイミング合わせおよびカウンター始動 |
| **ダッシュ** | Qキー | 攻撃の回避または間合いを詰める |
| **ダッシュ（スプリント）** | Shiftキー | ニュートラル状態での位置取り直し |

<Callout type="warning" title="スタミナ管理">
スタミナバーを完全に使い切らないようにしてください。攻撃を受けている最中にスタミナが切れると、ダッシュもパリィもできなくなり、相手のフルコンボをそのまま受ける「無防備な標的」になってしまいます。
</Callout>

</Section>

<Section icon={Phone} title="スマートフォンシステムとワールドインタラクション">

学ランは、スマートフォンシステムを介した非常に詳細な「ゲーム内ゲーム」を搭載しています。**Altキー**を押してスマートフォンを取り出し、**Mキー**を押してメニューを開くことで、さまざまなソーシャルアプリや機能アプリにアクセスできます。このシステムはゲームのロールプレイ（RP）要素において極めて重要であり、フレンドと連絡を取り合ったり、ダウンタイム中にミニゲームを楽しんだりすることができます。

スマートフォンシステムには以下が含まれます：
- **連絡先＆ダイヤラー**: 固有のIDを使用して、サーバー上の他のプレイヤーに電話をかけます。
- **メッセージ送信**: テキストを送信して、バスケットボールコートやクラブでの待ち合わせを調整します。
- **ミニゲーム**: キャラクターのスマホ画面上で直接、リズムベースのゲーム（Friday Night Funkinに類似）をプレイできます。

| アプリ名 | 機能 | 操作方法 |
| :--- | :--- | :--- |
| **ダイヤラー** | フレンドに電話をかける | IDを入力して通話ボタンを押す |
| **ゲーム** | リズムミニゲーム | 矢印キーまたはWASDを使用 |
| **連絡先** | ソーシャルリスト | クイックアクセスのためにプレイヤーIDを保存 |
| **設定** | スマホのカスタマイズ | 壁紙や着信音の変更 |

スマートフォンに加えて、ワールドにはインタラクティブなオブジェクトが溢れています。音楽室には、実際に演奏可能なピアノとギターが置かれています。ピアノの前で**Gキー**を押すと専用のインターフェースに入り、特定のキー（C、X、M、および数字キー）を使って実際のメロディを奏でることができます。

</Callout>
