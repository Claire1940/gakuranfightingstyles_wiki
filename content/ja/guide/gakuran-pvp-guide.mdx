<Section icon={Swords} title="戦闘操作の基本と基礎">

**学ランPvPガイド**をマスターするには、まず操作体系を深く理解することから始まります。一般的なボタン連打ゲームとは異なり、学ランでは意図的な入力と空間認識が求められます。戦闘システムは、正確なタイミングで実行する必要がある弱攻撃、強攻撃、および防御動作の基礎の上に構築されています。

| カテゴリ | 入力 | アクション | 用途 |
| :--- | :--- | :--- | :--- |
| **戦闘** | 左クリック | 基本攻撃 | コンボ構築のための標準的な近接攻撃 |
| **戦闘** | 右クリック | サブアクション | 強攻撃または独自の防御技 |
| **移動** | W, A, S, D | 移動 | 戦闘中の基本的な移動と横移動 |
| **移動** | Shift | ダッシュ | 素早い位置取りと距離管理 |
| **移動** | Space | ジャンプ | 垂直移動と低い攻撃の回避 |
| **インタラクション** | E | インタラクト | NPCや環境オブジェクトとの関わり |
| **インターフェース** | 電話キー | 電話を開く | メニュー、進行状況、設定へのアクセス |

戦闘のリズムは、相手の「予備動作（スタートアップ）」アニメーションを読み取る能力によって決まります。すべての攻撃には、カウンターや回避が可能な短い隙が存在します。現在の2026年のメタでは、防御の基礎を重視するプレイヤーが、攻撃的なスタイルのみに頼るプレイヤーを上回ることがよくあります。

<Callout type="tip" title="プロのヒント：カメラ管理">

</Callout>

<Section icon={Shield} title="高度なPvPメカニズムと防御">

**学ランPvPガイド**で真に卓越するには、基本攻撃を超えて、防御の三要素である「ガード（Blocking）」、「回避（Dodging）」、「間合い（Spacing）」をマスターしなければなりません。これらのメカニズムにより、ダメージを無効化し、相手が隙を見せる「パニッシュ（反撃）」のチャンスを作り出すことができます。

<Steps>
  <Step num="1" title="ガードを極める">
    防御入力を押し続けて、受けるダメージを軽減します。体幹（ポスチャー）ゲージを注意深く監視してください。ゲージが切れるとスタン状態になり、フルコンボを受ける隙が生まれます。
  </Step>
  <Step num="2" title="反応的な回避">
    移動キーとダッシュを組み合わせて、重攻撃をサイドステップで避けます。回避に成功すれば、カウンター攻撃に最適な位置を取れることが多いです。
  </Step>
  <Step num="3" title="間合いとフットワーク">
    自分が攻撃を当てられ、かつ相手の攻撃が届かない距離を保ちます。この「スイートスポット」は、使用している格闘スタイルの射程によって異なります。
  </Step>
  <Step num="4" title="コンボの実行">
    M1攻撃を特殊技や重攻撃のフィニッシャーに繋げます。リスクの高い学校での戦いに挑む前に、安全地帯でタイミングを練習しましょう。
  </Step>
</Steps>

| 防御アクション | 主なメリット | リスク要因 |
| :--- | :--- | :--- |
| **通常ガード** | 弱ダメージを無効化 | ガード崩しに弱い |
| **ジャストパリィ** | 攻撃者をスタンさせる | フレーム単位の正確なタイミングが必要 |
| **バックダッシュ** | ニュートラルな状態に戻す | スタミナを大幅に消費する |
| **サイドステップ** | 側面を突く隙を作る | 多用すると読まれやすい |

<Callout type="info" title="スタミナ管理">
スタミナは最も貴重なリソースです。すべてのダッシュとジャンプで消費されます。戦闘中にスタミナが切れると回避ができなくなり、高ダメージのフィニッシャーの格好の標的になります。
</Callout>

</Section>

<Section icon={Target} title="戦闘を支配するための戦略的ヒント">

学ランで勝つためには、単にクリックが速いだけではなく、より賢い意思決定が必要です。上級プレイヤーは心理戦を駆使して、相手を誘い出しミスを誘発させます。よくある間違いを理解することで、自分自身のミスを防ぎ、相手の隙を突くことができます。

<CardGrid cols={2}>
  <Card title="忍耐強いアプローチ">
    相手が先に動くのを待ちます。相手の動きに反応してパリィや回避を行い、相手が硬直（リカバリーフレーム）している間に確定コンボを叩き込みます。
  </Card>
  <Card title="攻撃的なプレッシャー">
    絶え間ない動きと牽制（弱攻撃）で相手にガードを強要します。相手の体幹が低くなったら、重攻撃を繰り出してガードを崩し、一気に決着をつけます。
  </Card>
</CardGrid>

| よくある間違い | 結果 | 改善方法 |
| :--- | :--- | :--- |
| **ボタン連打** | パリィされやすくなる | 意図を持ってクリックし、敵を観察する |
| **過度なダッシュ** | 防御用のスタミナが枯渇 | 距離を詰めたり離したりするときだけ使う |
| **射程の無視 | 攻撃が頻繁に空振る | 自分のスタイルの正確なリーチを覚える |
| **予測可能な回避** | 範囲攻撃に捕まる | 回避の方向とタイミングに変化をつける |

<Callout type="tip" title="プロのヒント：環境の活用">

</Callout>
