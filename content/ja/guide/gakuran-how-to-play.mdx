<Section icon={Swords} title="戦闘スタイルとレアリティ">

Gakuranの心臓部はその多様な戦闘スタイルにあります。各スタイルはユニークな特典やアニメーションを持っており、どのように戦闘に取り組むかを決定します。スタイルはレアリティによって分類されており、一般的からレアまであります。現在、メタはキャラクターの身長と範囲を補完するスタイルを重視しています。

利用可能な戦闘スタイルには以下があります：
- **ムエタイ**：強力なストライクと膝/肘の組み合わせに焦点を当てています。
- **アカリ**：現在のトップチームのスタイルで、優れたフローとダメージを提供しています。
- **空手**：バランスの取れたスタイルで、訓練されたストライクに焦点を当てています。
- **レスリング**：絞り合いと近距離優位性に焦点を当てています。
- **ボクシング**：高速のパンチで上半身の動きに焦点を当てています。
- **基本**：すべての新規プレイヤーのためのスタートスタイルです。

| 戦闘スタイル | 稀有度 | 主な強み |
| :--- | :--- | :--- |

| **ボクシング** | 単純 | M1の速さが高く、回復が早い |
| **レスリング** | 単純 | 守りを破る能力が強い |
| **空手** | 通常 | 初心者にも簡単に学べる信頼性が高い |

<Section icon={Shield} title="戦闘マスターと技術">

Gakuranで成功するためには、単なるボタンの押し続けを超えて動く必要があります。戦闘システムはタイミング、スペース、心理戦を報酬とします。ベテランプレイヤーとの遭遇を生き延びるためには、「カウンター-コンボ」サイクルを理解することが不可欠です。

**カウンターの技術**
カウンターは、敵の攻撃が当たる直前に**F**キーを押して行います。成功したカウンターは攻撃者を一時的に脆弱にし、自分のコンボを始めることができます。ブロックボタンを永遠に押さないでください、重い攻撃や特定のコンボによってガードが破られる可能性があります。

**高度なコンボの文字列**
新規プレイヤーの間で一般的な間違いは、M1（左クリック）ボタンをスパムすることです。これにより攻撃が予測可能になり、カウンターしやすくなります。代わりに以下の技術を使用してください：
- **1-2-クリティカル**：M1を2回ヒットし、すぐにクリティカル攻撃を続けます。これは通常、3回目のM1ヒットをカウンターしようと待っているプレイヤーを捕まえます。
- **ダッシュ（Q）**：Qキーを使用して後退または横方向にダッシュします。カウンターが失敗した場合に位置をリセットするための非常に重要な技術です。
- **スペース**：自分と敵の間に少し距離を保ちます。攻撃を始める準備ができたらだけギャップを縮めます。

| 技術 | 入力 | 戦略的な使用 |
| :--- | :--- | :--- |
| **M1スパム** | 左クリック | ガードを破れたり、スタンディング状態の敵に対してのみ使用 |
| **クリティカル** | やり方 | ガードを破ったり、コンボを終えるために使用 |
| **ダッシュ** | Qキー | 逃げたり、距離を縮めるための非常に重要な技術 |
| **スプリント攻撃** | シフト + M1 | 遠距離から戦闘を始めるのに適しています |

<Callout type="warning" title="防禦回避戦術">
あなたの対戦相手がすべてを防禦していると感じたら、最初のヒットの後で連続攻撃を止め、逃げて行ってください。これにより、戦闘の「リズム」がリセットされ、彼らがあなたに対してカウンター攻撃を無償で行うのを防ぎます。
</Callout>

</Section>

<Section icon={Sparkles} title="電話システムとソーシャル機能">

Gakuranは戦闘ゲームと同様に、社会的な体験でもあります。開発者はスマートフォンインターフェースを通じて非常に詳細な「ゲーム内ゲーム」を含めています。**Alt**または**M**を押すことで、デバイスを取り出してさまざまなアプリと対話できます。

**電話機能：**
- **連絡先・通話**：友人や最近のプレイヤーに電話してミーティングやチャレンジを調整します。
- **ミニゲーム**：電話には実際に遊べるゲームがあり、友達を待っている間や戦闘の間に楽しめます。
- **設定**：個人の好みや社会的な可視性を調整します。

電話の他に、学校環境にはいくつかのインタラクティブなエリアがあります：
- **音楽室**：本館内に位置し、ピアノとギターがあります。**G**を押してピアノと対話し、人気の音楽タイトルに似たリズムベースのミニゲームがトリガーされます。
- **バスケットボールコート**：時々マップバグに悩まされるものの、コートはプレイヤーがボールを拾い、**F**を押してシュート練習ができます。
- **RPサーバー**：パブリックスーバーはしばしば混沌としていますが、「学生生活」の側面に焦点を当てたい人々にとって、構造化された環境を提供する専用のロールプレイ（RP）サーバーがあります。
