戦闘システムは攻撃入力と防御入力に分かれています。パンチを繰り出す前に、まず**T**キーを押して戦闘スタンスを有効にする必要があります。スタンスが有効になると、キャラクターは構えの姿勢に移行し、すべての格闘技アクションが使用可能になります。

| 入力 | アクション | 機能と戦略 |
| :--- | :--- | :--- |
| **T** | スタンス切り替え | 戦闘を開始し、すべての攻撃アクションにアクセスするために必要 |
| **M1 (左クリック)** | 通常攻撃 | 素早い打撃。最大4ヒットのコンボをつなげることが可能 |
| **R** | ヘビーアタック | ガード不能の打撃。2倍のダメージを与え、相手をラグドール化させる |
| **F** | ブロック | 受けるダメージを軽減するが、姿勢（ポスチャー）バーを消費する |
| **Q** | 回避 | 無敵時間（I-frame）を利用して位置を変えたり、重い一撃を避けたりする |
| **Ctrl** | ユーティリティメニュー | ミニゲーム、写真撮影、ソーシャルエモートへのアクセス |

<Section icon={Target} title="攻撃アクションとコンボの実行">

**gakuranの操作**における攻撃面は、弱攻撃と強攻撃の相乗効果を中心に展開します。相手にプレッシャーを与え続けるためには、M1コンボのマスターが不可欠です。標準的なM1シーケンスは4連続の打撃で構成されています。このシーケンスの最後の一撃は特殊な蹴りであり、強力なノックバック効果を与えます。このノックバックは、追撃のチャンスを作ったり、体力が低い場合に態勢を立て直したりするために非常に重要です。

**R**キーに割り当てられているヘビーアタックは、ブロックを多用するプレイヤーに対処するための主な手段です。これらの攻撃は標準的なM1の約2倍のダメージを与えますが、動作が大幅に遅くなります。この予備動作の長さにより、パリィ（受け流し）を狙われやすくなります。そのため、ヘビーアタックは戦略的に使用する必要があります。通常は、相手を追い詰めた時や、回避に成功した直後に繰り出すのが効果的です。

| 攻撃タイプ | ダメージ出力 | ノックバック力 | ガード可否 |
| :--- | :--- | :--- | :--- |
| **M1コンボ (1-3打目)** | 低〜中 | 最小限 | ガード可能 |
| **M1フィニッシャー (4打目)** | 中 | 高（蹴り） | ガード可能 |
| **ヘビーアタック (R)** | 非常に高い | 極大（ラグドール） | ガード不能 |
| **パニックアタック** | 変動 | 低 | 連打可能 |

標準的な打撃に加えて、プレッシャーのかかる状況でM1を素早く連打することで「パニックアタック」を実行できます。技術的ではありませんが、相手のリズムを崩すことができる場合があります。しかし、これに頼りすぎると、パーフェクトブロックのタイミングを理解している熟練プレイヤーに対して隙を見せることになります。

<Callout type="warning" title="ヘビーアタックのリスク">
**R**キーのヘビーアタックは動作が遅いため、経験豊富なプレイヤーはアニメーションが始まるのを待ってから、即座にパーフェクトブロックを試みてきます。高スキルの相手に対しては、ヘビーアタックの使用を控えめにしましょう。
</Callout>

</Section>

<Section icon={Shield} title="防御システムとパーフェクトブロック">

現在の**gakuranの操作**メタにおいて、防御は攻撃よりも重要であると言えます。このゲームでは**姿勢（ポスチャー）バー**システムが採用されており、ブロック中にどれだけの攻撃を吸収できるかが決まります。**F**キーを押し続けると姿勢バーが急速に減少し、空になると「ガードブレイク」が発生します。これにより2秒間のスタン状態となり、完全に無防備になってしまいます。

姿勢の消耗を避けるためには、**パーフェクトブロック**をマスターする必要があります。このメカニクスは、攻撃が当たる直前の0.2秒以内に**F**キーを押すことで発動します。パーフェクトブロックに成功すると、姿勢を全く消費せず、さらに攻撃者を0.5秒間スタンさせることができ、大きなアドバンテージを得られます。

<Steps>
  <Step num="1" title="攻撃パターンの観察">
    相手のアニメーションの予備動作を見て、軽いM1攻撃か重いR攻撃かを予測します。
  </Step>
  <Step num="2" title="入力のタイミング">
    打撃が当たる直前に**F**キーを押します。一貫性を保つには、低遅延（低Ping）の環境が理想的です。
  </Step>
  <Step num="3" title="カウンター攻撃">
    パーフェクトブロックの直後にM1コンボやスタイル固有の技を繰り出し、相手の短いスタン時間を活用します。
  </Step>
  <Step num="4" title="姿勢のリセット">
    姿勢バーが低くなったら、**Q**キーで回避して距離を取り、バーが自然に回復するのを待ちます。
  </Step>
</Steps>

回避メカニクス（**Q**）も同様に重要です。これは「無敵時間（I-frame）」を提供し、ダメージを受けることなく攻撃をすり抜けることができます。これは、通常ブロックできないヘビーアタックに対処する最も効果的な方法です。横に回避することで相手の「死角」に回り込み、背後からの攻撃のチャンスを作ることができます。

| 防御アクション | リソースコスト | 主な利点 |
| :--- | :--- | :--- |
| **通常ブロック** | 高い姿勢消費 | 安全なダメージ軽減 |
| **パーフェクトブロック** | 消費なし | 敵をスタンさせ、全ダメージを無効化 |
| **回避 (Q)** | スタミナ/クールタイム | 無敵時間と位置取りの変更 |
| **回復 (保健室)** | 時間 | 体力回復速度が3倍になる |

<Callout type="success" title="パーフェクトブロックの報酬">
ボクシングスタイルを使用している場合、パーフェクトブロックに成功するとヘビーアタックのクールタイムがわずか1秒に短縮され、非常に攻撃的なカウンタープレイが可能になります。
</Callout>

</Section>

戦闘には2つの隠しステータスも関わっています：**ガード貫通（Guard Pierce）**と**強靭さ（Resilience）**です。ガード貫通は、相手がブロックしていても「削りダメージ」を与えることができます。強靭さは、2人のプレイヤーが同時にM1を当てた時に発生する「掴み合い（グラップル）」で勝つ確率を高めます。強靭さが高いほど、これらの掴み合いから素早く抜け出し、カウンターを仕掛けることができます。

| **アフリカ系** | 1% | 体力/ダメージ | タンク/ドミネーター |
| **ヨーロッパ系** | 1% | ダメージ/リーチ | ヘビーヒッター |
| **インド系** | 1% | スピード/俊敏性 | スカーミッシャー |

<Section icon={Sparkles} title="高度な格闘スタイルとメタの選択">

基本的な**gakuranの操作**をマスターしたら、戦闘能力を特化させるために格闘スタイルを選択する必要があります。現在、ゲームには8つのユニークなスタイルがありますが、2026年の「メタ」として際立っているのは以下の2つです。

**ボクシングスタイル**
ボクシングは、低身長および中身長のプレイヤーにとって最高の選択肢です。極めて攻撃的なプレイと無敵時間の活用に焦点を当てています。このスタイルのヘビーアタックは対策が非常に難しく、相手に回避のクールタイムを無駄遣いさせることがよくあります。パーフェクトブロック時にヘビーアタックのクールタイムが短縮されるため、ボクサーは戦場で絶え間ない脅威となります。

**レスリングスタイル**
レスリングは、高身長プレイヤーにとって支配的な選択肢です。そのヘビーアタックは相手を掴んで地面に叩きつけるもので、一度始まると中断させることができません。さらに、レスリングはゲーム内で最高の強靭さステータスを提供するため、レスラー以外のプレイヤーとの掴み合いで負けることはほぼありません。
