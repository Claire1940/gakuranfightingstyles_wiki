<Section icon={Keyboard} title="移動操作の基本">

**学ランの戦闘操作**を極めるには、スムーズな移動から始まります。2007年の日本の学校をイメージした「学ラン」の世界では、街中や校舎の廊下をいかに動き回るかが、戦闘の開始や離脱を左右します。一般的なロールプレイングゲームとは異なり、本作の移動は慣性（モメンタム）に基づいているため、停止や方向転換にはわずかな重みがあります。

| 入力 | アクション | 用途 |
| :--- | :--- | :--- |
| **W, A, S, D** | 移動 | 学校や街の中の基本的な移動 |
| **Shift** | ダッシュ | 探索や追跡時の移動速度アップ |
| **Space** | ジャンプ | 障害物を飛び越える、または垂直方向のポジショニング |
| **マウス** | カメラ | 視点を回転させ、360度全方位で相手を追跡 |
| **Esc** | メニュー | Robloxの設定にアクセスして感度を調整 |

<Callout type="tip" title="プロのコツ：移動バフ">
側面から回り込んでくる敵を察知できるよう、カメラは常にアンロック状態にしておきましょう。「学ラン」では、不良に不意を突かれると、戦いが始まる前に勝負が決まってしまうこともあります。戦闘中は**Shift**を控えめに使い、目に見えないスタミナゲージを管理しましょう。
</Callout>

効果的な移動とは、単にA地点からB地点へ行くことだけではありません。重要なのは「スペーシング（間合いの管理）」です。移動キーを押し続けるのではなく、タップするように入力することで、微調整が可能になり、相手が最初のM1（マウス左クリック）コンボを当てにくくなります。 

</Section>

<Section icon={Swords} title="中心的な戦闘メカニクスと入力">

このゲームの醍醐味は、その戦闘システムにあります。中庭で自分の威厳を守るためであれ、組織化されたPvPに挑むためであれ、主要な攻撃入力を理解することは必須です。**学ランの戦闘操作**は標準的なM1/M2システムを採用していますが、タイミングやスタイル固有の補正によって奥深さが加わっています。

| カテゴリ | 入力 | アクション | プロのタイミングのコツ |
| :--- | :--- | :--- | :--- |
| **攻撃** | **左クリック (M1)** | 弱攻撃 | 3〜4回連続でヒットさせて基本コンボを繋ぐ |
| **強攻撃** | **右クリック (M2)** | 強攻撃 | ガード（ブロック）崩しやコンボのフィニッシュに使用 |
| **インタラクト** | **E** | インタラクト | 倒れた相手にトドメを刺す、またはNPCと会話する |
| **ユーティリティ** | **F (標準)** | ブロック | 長押しで受ける弱攻撃のダメージを軽減 |

<Callout type="warning" title="戦闘時の注意">
左クリックを連打しないでください。「ボタン連打」はキャラクターがアニメーションロック（硬直状態）に陥りやすく、3打目を相手に回避またはブロックされた場合、手痛い反撃を受ける隙を晒すことになります。
</Callout>

<Section icon={Shield} title="防御アクションと生存戦略">

「学ラン」での生存は、防御に対する理解度にかかっています。**学ランの戦闘操作**において攻撃ばかりに気を取られていると、受け流し（パリィ）や回避を駆使する熟練プレイヤーを相手に苦戦することになります。防御は能動的なものであり、敵の攻撃リズムを予測する必要があります。

<Steps>
  <Step num="1" title="初期ガード">
    相手が近づいてくるのが見えたら、ブロックキーを長押しします。これにより弱攻撃による削りダメージを抑え、完全なよろけ状態を防ぐことができます。
  </Step>
  <Step num="2" title="強攻撃の読み">
    「予備動作」のアニメーションに注目しましょう。強攻撃には通常、独特の視覚的な合図があります。これが回避するか、あるいはカウンターの準備をするかのシグナルです。
  </Step>
  <Step num="3" title="カウンターダッシュ">
    ジャンプやダッシュと組み合わせて移動キーを使い、相手の強攻撃を「すり抜ける」ように回避し、即座に自身のM1コンボで追撃します。
  </Step>
</Steps>

<Callout type="success" title="ジャストガード（パーフェクトブロック）の仕組み">
タイミングを合わせたブロック（攻撃が当たる瞬間のブロック）は、高確率で相手をよろけさせ、コンボを叩き込む絶好のチャンス（無償コンボの窓）を作り出します。リスクの高いPvPに挑む前に、NPCを相手にこのタイミングを練習しておきましょう。
</Callout>

| 防御アクション | 操作方法 | メリット | リスク |
| :--- | :--- | :--- | :--- |
| **通常ブロック** | キー長押し | ダメージを70〜80%軽減 | M2（強攻撃）でガードを崩される |
| **回避ダッシュ** | Shift ＋ 方向キー | ダメージを完全に無効化 | スタミナ/慣性を大きく消費する |
| **間合いの管理** | Sキー（後退） | M1の射程外を維持 | 遠距離スタイルに弱い |

</Section>

<Section icon={Sparkles} title="格闘スタイルとコンボの実行">

選択した格闘スタイルによって、**学ランの戦闘操作**の感覚は大きく変化します。スピードと多段コンボを重視するスタイルもあれば、純粋なパワーと投げ（グラップル）に特化したスタイルもあります。2026年のメタでは、ニュートラルな立ち回りとパニッシュ（反撃）の機会の双方で汎用性の高いスタイルが好まれています。

<CardGrid cols={2}>
  <Card title="ストライカースタイル（速度重視）">
    - M1の硬直回復が早い
    - 1撃あたりのダメージは低い
    - 「ヒット・アンド・アウェイ」戦術に最適。
  </Card>
  <Card title="パワースタイル（力重視）">
    - M2のダメージが非常に大きい
    - 移動速度が遅い
    - 攻撃を耐えつつ、ダメージを相打ちにするのに最適。
  </Card>
</CardGrid>

<Callout type="info" title="スタイルの相乗効果">
特定のキャラクター特性（トレイト）によって、スタイルの操作性を向上させることができます。例えば、「迅速（Quick）」の特性は強攻撃の予備動作を短縮し、相手がM2に反応するのを非常に困難にします。
</Callout>

基本コンボを実行するには、以下の手順に従ってください：
1. **交戦（エンゲージ）**: Shift + W でダッシュして接近します。
2. **セットアップ**: M1攻撃を3回当てます（左クリック×3）。
3. **フィニッシャー**: 即座にM2（右クリック）を押し、相手を吹き飛ばします。
4. **リセット**: 一度下がってスタミナを回復させ、次の隙をうかがいます。

</Section>

<Section icon={Trophy} title="高度なPvP戦略とマスターへの道">

**学ランの戦闘操作**の基本を超えてステップアップするには、PvPにおける「心理戦（メンタルゲーム）」をマスターする必要があります。「学ラン」での戦闘は、ボタンを押すことと同じくらい、心理学的な駆け引きが重要です。相手を誘い出し（ベイト）、防御用のクールダウンを無駄遣いさせることが、上位プレイヤーに勝利するための鍵となります。

**高度な戦術：**
- **ベイト（誘い出し）**: 相手に近づいてから、突然下がります。多くの場合、相手は空振りのM1やM2を繰り出すため、その隙を突いて反撃（ウィフパニッシュ）を狙えます。
- **ガードプレッシャー**: 相手がブロックを維持している場合、M1をただ連打するだけではいけません。M2を織り交ぜて相手のガードを崩し、体勢（ポスチャー）に大きなダメージを与えましょう。
- **環境の意識（ステージの活用）**: 学校の狭い廊下を利用して動きの速い相手を追い詰めたり、広い中庭を利用して動きの遅い「パワー」スタイルの周囲を回り込むように立ち回ったりします。

| 戦略 | 有効性 | 難易度 | 最も効果的な相手 |
| :--- | :--- | :--- | :--- |
| **ウィフパニッシュ（空振りへの反撃）** | 高 | 難 | 攻撃的な「メーカー（手数の多いプレイヤー）」 |
| **ガード崩し** | 中 | 易 | 防御的な「タートル（引きこもりプレイヤー）」 |
| **スタミナ追跡** | 極めて高 | 中 | すべてのスタイル |

<Callout type="success" title="マスターへの道">
2026年の「学ラン」メタにおいて、一貫性（継続性）こそが絶対の正義です。毎セッション10分間、移動とブロックのタイミングを練習しましょう。やがてマッスルメモリー（身体の記憶）が定着し、自分の操作キーではなく、相手の行動パターンに集中できるようになります。
</Callout>

</Section>
