<Section icon={Shield} title="Guia de Mecânicas de Combate e Controles">

Para utilizar os **melhores estilos de luta de Gakuran** de forma eficaz, você deve dominar o esquema de controles. O jogo usa um layout padrão do Roblox, mas adiciona camadas de complexidade através das interações de "Shift-Lock" e "Bloqueio".

| Categoria | Comando | Ação | Dica Pro |
| :--- | :--- | :--- | :--- |
| **Movimento** | **W, A, S, D** | Movimento Básico | Mova-se em círculos para evitar golpes lineares |
| **Correr** | **Shift Esquerdo** | Movimento Rápido | Use para atrair o ataque pesado de um inimigo |
| **Ataque** | **Mouse Esquerdo (M1)** | Ataque Leve | Encadeie 3 golpes e depois pause ou use M2 |
| **Pesado** | **Mouse Direito (M2)** | Ataque Pesado | Quebra bloqueios, mas é facilmente esquivado |
| **Interagir** | **E** | Interação | Use para movimentos de finalização ou ambiente |
| **Defesa** | **F (Segurar)** | Bloqueio | Reduz o dano; o timing cria aberturas |

Um combate eficaz envolve "ler" seu oponente. Se ele estiver segurando o bloqueio, não spamme ataques M1; em vez disso, use um ataque pesado M2 para quebrar a guarda e iniciar seu combo completo.

<Callout type="info" title="Consciência de Combate">
Sua câmera é sua melhor arma. Use o mouse para rastrear oponentes mesmo quando eles derem um dash para trás de você. Perder o contato visual por um segundo geralmente resulta em uma barra de vida perdida.
</Callout>

</Section>
